# e-plantShopping

## Project Name: E-Plant Shopping Application

### Overview
Welcome to **e-plantShopping**, an e-commerce web application designed for plant enthusiasts. This application allows users to browse a wide selection of indoor and outdoor plants, view detailed care instructions, add items to a shopping cart, and seamlessly manage their prospective purchases before checking out. 

### Features
* Browse plants by category
* Interactive shopping cart with real-time price updates
* Responsive UI designed with React
